# ❓ Foire Aux Questions (F.A.Q)

**Q : Comment récupérer mon véhicule à la fourrière ?**
R : Rendez-vous au point de fourrière sur votre carte, payez les frais et votre véhicule apparaîtra.

**Q : Je suis bloqué dans le décor, que faire ?**
R : Utilisez la commande `/report` pour qu'un staff vienne vous débloquer.

**Q : Où se trouve le Pôle Emploi ?**
R : Le logo "Valise" sur votre carte, au centre-ville.
